KonovalovApi - сделанный CustomerApi для CRUD-операции.
Чтобы использовать программу необходимо:
- Используя браузер перейти на http://localhost:5023 или на https://localhost:7268
- Перед тем как использовать программу необходимо перейти во вкладку регистрации и зарегистрироваться: Ввести E-mail, пароль и имя
 (имя необходимо ввести, несмотря на то, что указано необязательно иначе будет ошибка)
- Затем необходимо перейти на страницу входа и войти в учетную запись
- Готово, теперь можно производить CRUD-операции.
